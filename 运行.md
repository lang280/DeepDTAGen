```bash
cd /home/jianlang/DeepDTAGen

```

---

## 各脚本运行方式

### 1. 数据预处理（已完成，可跳过）

```bash
python create_data.py
```

生成 `data/processed/` 下的 `.pt` 文件。

---

### 2. 训练模型

脚本需要传入数据集索引，对应关系：

| 参数 | 数据集    |
|------|-----------|
| `0`  | davis     |
| `1`  | kiba      |
| `2`  | bindingdb |

```bash
# 用 CPU 训练（第一个参数为数据集索引）
python training.py 0   # davis
python training.py 1   # kiba
python training.py 2   # bindingdb

# 用 GPU 0 训练（第二个参数为 GPU 编号）
python training.py 0 0
```

---

### 3. 测试预训练模型

预训练权重在 `models/` 目录下，直接运行：

```bash
python test.py
```

---

### 4. 生成药物

```bash
python generate.py bindingdb --device cpu
python generate.py davis --device cpu
python generate.py kiba --device cpu
```

### 5. 评估生成结果

```bash
python generation_eveluation.py
```
