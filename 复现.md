## 1. 本地虚拟环境

- `.venv/`
  - 来源：2026-04-15 在仓库根目录用 `python3.8 -m venv .venv` 创建。
  - Python 来源：系统安装的 `python3.8` / `python3.8-venv`，由 `sudo apt-get install -y libarchive-tools unrar-free python3.8 python3.8-venv python3.8-distutils` 安装。
  - 当前版本：`.venv/pyvenv.cfg` 记录为 `3.8.20`。
  - 包来源：根据仓库中的 [environment.yml](/home/jianlang/DeepDTAGen/environment.yml) 和 README 安装，并补齐 PyG CPU 轮子。
  - 主要安装命令：

```bash
python3.8 -m venv .venv
.venv/bin/python -m pip install --upgrade pip setuptools wheel
.venv/bin/python -m pip install numpy==1.23.5 pandas==1.5.2 scipy scikit-learn tqdm networkx einops==0.6.0 rdkit-pypi==2022.9.5 torch==1.12.1+cpu torchvision==0.13.1+cpu torchaudio==0.12.1+cpu --extra-index-url https://download.pytorch.org/whl/cpu fairseq==0.10.2
.venv/bin/python -m pip install torch-scatter==2.1.0+pt112cpu torch-sparse==0.6.16+pt112cpu torch-cluster==1.6.0+pt112cpu torch-spline-conv==1.2.1+pt112cpu -f https://data.pyg.org/whl/torch-1.12.0+cpu.html
.venv/bin/python -m pip install torch-geometric==2.2.0
```
